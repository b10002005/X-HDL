About X-HDL System Overview

Hello, It's X-HDL!

This is a intelligent online tool which is designed for HDL designers. It provides powerful capabilities to help designers to quickly understand the function of the circuit as well as understanding the process of the circuit. It only need to do is upload your HDL design amd with some simple click. Then, you can do any operation and watching the results of the analysis on X-HDL.
